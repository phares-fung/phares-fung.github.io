Git下载安装（Windows）

2020-08-19 1 分钟阅读 #工具#软件安装#Git

本文展示了如何安装 Git。

下载Git

下载链接：Git官网下载 ，根据自己的电脑操作系统来选择下载

进行安装

下载完之后，双击进行安装

选择安装目录

选择组件

开始菜单目录名设置

选择使用命令行环境

后续默认，点击下一步

安装完成

检验是否安装成功

回到桌面，鼠标右键点击，会出现Git GUI Here和Git Bash Here

进入CMD命令窗口，输入git，显示有git相关信息（如下图），则表明安装成功
