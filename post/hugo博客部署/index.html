Hugo博客部署

Jul 27, 2021
1 min read

1. 创建仓库

在github上，创建两个仓库，均为master分支：

  1. username.github.io // username为你自己github的用户名，存放编译后的静态文件
  2. blog // 名称随意，存放源文件

2. 推送源文件

在站点根目录，先将编译的静态文件夹public删除，再执行以下命令：

git init
git remote add origin git@github.com:phares-fung/blog.git
git add -a
git commit -m "first commit"
git push -u origin master

此时，就将站点源文件上传到github上了。

3. 推送静态文件

在站点根目录，执行命令，拉取子模块username.github.io到public目录：

git submodule add -b master git@github.com:phares-fung/phares-fung.github.io.git public

之后执行命令，生成静态文件：

hugo -t m10c

最后执行提交命令，将所有文件改动提交到仓库。

此时可以通过git page网站 https://username.github.io/ 查看博客站点。

4. 新环境使用

  1. 安装hugo
  2. 从仓库拉取源文件
  3. 修改后执行生成静态文件的命令
  4. 提交所有更改
  5. 访问git page网站查看
